# Structural Design Patterns

This project contains implementations of various Structural Design Patterns using Django.

## Patterns Included

1.  **Adapter**: `adapter`
2.  **Bridge**: `bridge`
3.  **Composite**: `composite`
4.  **Decorator**: `decorator`
5.  **Facade**: `facade`
6.  **Proxy**: `proxy`

## Setup

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run migrations:
    ```bash
    python manage.py migrate
    ```
3.  Run the server:
    ```bash
    python manage.py runserver
    ```
