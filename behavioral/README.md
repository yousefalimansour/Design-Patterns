# Behavioral Design Patterns

This project contains implementations of various Behavioral Design Patterns using Django.

## Patterns Included

1.  **Observer**: `observer`
2.  **Strategy**: `strategy`
3.  **Command**: `command`
4.  **Template Method**: `template_method`
5.  **Iterator**: `iterator`
6.  **State**: `state`
7.  **Chain of Responsibility**: `chain_of_responsibility`

## Setup

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run migrations:
    ```bash
    python manage.py migrate
    ```
3.  Run the server:
    ```bash
    python manage.py runserver
    ```
